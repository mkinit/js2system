# JS2system 前端SSR应用
基于nuxt

## 本地运行项目
- 安装项目依赖：npm install
- 启动项目：npm run dev
- 默认地址：http://localhost:3000

服务器部署需要先编译项目：npm run build，然后再启动：npm run start
